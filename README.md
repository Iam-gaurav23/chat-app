# ChatApp – Real-Time Messaging Platform

## 🚀 Overview
A full-stack real-time chat application built using the MERN stack.  
This application allows users to communicate instantly with features like authentication, online status, and real-time messaging.

---

## ✨ Features
- Real-time messaging using Socket.IO
- Secure authentication using JWT
- Online/offline user status
- Responsive UI using Tailwind CSS
- (Add your custom feature here 👈)

---

## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS, Zustand
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time: Socket.IO
